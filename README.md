# Come-y-Calla 🍲

¡Bienvenido al proyecto **Come-y-Calla**! 
Esta es una *landing page* / invitación interactiva diseñada para un evento gastronómico o menú digital. Cuenta con un diseño divertido, animaciones fluidas y una estética de "caricatura 3D" con colores cálidos para despertar el apetito.

## 🎨 Características de Diseño

El proyecto utiliza una interfaz de usuario personalizada basada en la temática **"Chef Clásico"**:

- **Efectos 3D y Caricatura:** Uso de sombras pronunciadas (`box-shadow` / `drop-shadow`) y bordes muy redondeados (`25px`) para dar un aspecto amigable y moderno.
- **Animaciones:** El personaje principal ("Tobyas") cuenta con una animación flotante continua y los botones tienen efectos de crecimiento y brillo al pasar el cursor (hover).
- **Tipografía:** 
  - `Fredoka` para encabezados y botones (aspecto redondeado y divertido).
  - `Quicksand` para el texto principal (limpia y de fácil lectura).

## 🧱 Estructura de la Página

La hoja de estilos está preparada para estilizar las siguientes secciones:
1. **Hero Section (`.hero`):** Sección principal de bienvenida con título animado y la imagen flotante de Tobyas.
2. **Invitation Card (`.invitation-card`):** Una tarjeta de invitación con bordes punteados simulando un ticket o carta especial.
3. **Mapa (`.map-wrapper`):** Contenedor para incrustar un iframe de Google Maps.
4. **Menú (`.menu-card`):** Tarjetas de comida con efecto 3D al hacer hover y *placeholders* para las imágenes de los platillos.
5. **Footer & Contacto (`.footer`, `.btn-action`):** Pie de página oscuro que hace resaltar el botón principal de acción (idealmente configurado para redirigir a WhatsApp).

## 🚀 Instalación y Uso

Al ser un proyecto estático compuesto por HTML y CSS:
1. Clona o descarga el repositorio.
2. Asegúrate de enlazar la hoja de estilos (`css/styles.css`) en tu archivo `index.html`.
3. Abre el archivo `index.html` en tu navegador web de preferencia para visualizar el proyecto.
